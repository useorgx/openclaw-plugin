# Twitter/X Ads Runbook (G4)

Tasks covered:
- Set up Twitter Ads account
- Define audience targeting
- Launch campaign with budget

Evidence should be captured as screenshots and written notes. Store in:
- `docs/ops/2026-02-14/twitter-ads/`

## 1) Account Setup

Checklist:
- Confirm billing profile and payment method.
- Confirm pixel / conversion tracking approach.
- Confirm account timezone and currency.

Evidence:
- Screenshot: billing settings
- Screenshot: account settings

## 2) Audience Targeting

Baseline targeting (adjust as needed):
- Geo: US + English
- Interests: software dev, developer tools, AI tooling
- Keywords: "MCP", "agent orchestration", "OpenAI", "Claude", "Cursor", "RAG", "LLM"
- Exclusions: existing customers (if list available)

Evidence:
- Screenshot: targeting definition
- Notes: rationale + expected CAC assumptions

## 3) Campaign Launch

Checklist:
- Objective: conversions (signup) or website clicks (if conversions not reliable yet)
- Budget: set daily spend cap + total cap
- Creative: at least 2 approved variants
- Landing: UTM-tagged

Evidence:
- Campaign ID
- Screenshot: campaign settings
- Screenshot: creatives list
- Screenshot/log: conversion tracking status
